Как это работает

От карточки клиента до PDF на руках — четыре шага

Карточка клиента

Имя, возраст, пол, симптомы и диагнозы, результаты анализов — всё в одном месте вместо блокнота.

Подбор по симптомам

Движок сопоставляет симптомы и диагнозы клиента с показаниями препаратов из каталога Siberian Wellness.

Программа по этапам

Очищение → Защита → Питание → Восстановление — с датами начала, длительностью курса и дозой по возрасту.

PDF клиенту

Готовая программа выгружается в PDF: список препаратов и шкала приёма на 60 дней.

Возможности

Всё, что нужно для приёма, а не для галочки

ДАННЫЕ

База клиентов

Карточка на каждого: возраст, пол, симптомы, диагнозы, история прежних программ.

КАТАЛОГ

Каталог Siberian Wellness

Состав, действие, показания и противопоказания каждого препарата — под рукой на приёме.

ДВИЖОК

Умный подбор

Симптомы и диагнозы клиента автоматически сверяются с показаниями препаратов каталога.

ДОЗЫ

Доза по возрасту

Отдельная дозировка для детей 6–12, 12–18 лет, взрослых и клиентов старше 65 — без пересчёта вручную.

ШКАЛА

График на 60 дней

Визуальная шкала приёма по этапам — сразу видно, что с чем сочетается и когда вводить следующий препарат.

PDF

Экспорт в один шаг

Готовая программа — печатный документ для клиента, без ручного набора текста после консультации.

Для консультанта

Десятки клиентов — и одна тетрадь на всех, если не Sibvaleo

  • Состав и противопоказания — в каталоге на том же экране
